Amphibians Portfolio Detail

The dataset is a multilabel classification problem. The goal is to predict the presence of amphibians species near the water reservoirs based on features obtained from GIS systems and satellite images. Here I used the Binary Relevance approach because I considered that all class labels are independent of each other. So, the occurrence of one class label does not influence the occurrence of another one. For this project, I used pandas to manipulate the data, matplotlib and seaborn to build a chart and sklearn to preprocess the data as a MultiLabelBinarizer and skmultilearn building a model with Binary Relevance. During the project we started to understand the data and the categories of the variables then we transformed the variables, built the model, and improved the memory usage. We looked for NAN values and unique values too and transform que category types to binary ones. The Total Hamming Loss of 0,26 and F1 Score of 0,69.
